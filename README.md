# AI-Minesweeper

Usage:

```shell
pip3 install -r requirements.txt
python3 runner.py
````

Coded according to the requirements listed on [Harvard CS50 AI website](https://cs50.harvard.edu/ai/2020/projects/1/minesweeper/), with several key improvements.
